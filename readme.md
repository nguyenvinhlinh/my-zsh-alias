# My ZSH Alias

## Clone source code
```
$ mkdir $HOME/Software;
$ cd    $HOME/Software;
$ git clone git@github.com:nguyenvinhlinh/my-zsh-alias.git my-zsh-alias
```

## Modify `.zshrc`
```
export MY_ALIAS=/home/nguyenvinhlinh/Software/my-zsh-alias;
source /home/nguyenvinhlinh/Software/my-zsh-alias/main.zsh;
```

## Test
